# Genesys Forge Client

React + Vite frontend skeleton for the Genesys Forge application.

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run test
```

## Structure

- `src/app` - app providers and routing.
- `src/features` - user-facing feature slices.
- `src/api` - API clients and shared TypeScript DTO mirrors.
- `src/stores` - local client state.
